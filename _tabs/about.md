## 关于我

我是 **Jamsread**，一名 C++ 客户端工程师。

### 技术栈

- **语言**：C++ / Python / Shell
- **框架**：Qt / Boost
- **领域**：桌面应用开发、NAS 工具、文件同步、Markdown 解析
- **工具**：Git / CMake / Docker

### 工作方向

我的日常工作围绕桌面端和 NAS 场景展开，主要关注：

- **性能优化**：针对大文件和高并发场景的 I/O 优化
- **跨平台兼容**：确保工具在 Windows / macOS / Linux 上的一致体验
- **用户体验**：用 Qt 构建简洁高效的图形界面

### 项目

| 项目 | 简介 |
|------|------|
| **Markdown Reader** | NAS 上的 Markdown 阅读器，支持离线浏览与全文搜索 |
| **Qt Editor** | 基于 Qt 的轻量级文本编辑器 |
| **Remote Sync** | 跨设备文件远程同步工具 |

### 开源理念

我相信好的工具应该对所有人开放。我的项目大多以开源形式发布在 GitHub 上，欢迎交流与贡献。

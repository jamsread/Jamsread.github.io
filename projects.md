## Markdown Reader for NAS

**NAS 上的 Markdown 阅读器**

一款部署在 NAS 上的轻量级 Markdown 阅读与管理工具，适合用来整理技术笔记和个人知识库。

- 支持 Markdown 实时渲染
- 全文搜索与标签分类
- 响应式 Web 界面，手机也能用

## Qt-based Editor

**基于 Qt 的轻量级编辑器**

用 C++ 和 Qt 构建的桌面文本编辑器，追求启动快、占用少。

- 语法高亮（Markdown / C++ / Python）
- 多标签页编辑
- 跨平台：Windows / macOS / Linux

## Remote Sync Tools

**远程文件同步工具**

解决多台设备间文件同步的痛点，支持增量传输和冲突检测。

- 基于自定义协议的增量同步
- 冲突检测与合并策略
- 命令行界面，适合自动化脚本集成
